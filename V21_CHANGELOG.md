# V21 Changelog

## 1. 個股「近五天新聞與當日重點」摘要改寫

- 改成事件脈絡式摘要，不再把新聞標題逐條貼上。
- AI prompt 要求先整理近五天主軸、今日是否有新事件、再用股價與量能作為輔助判讀。
- fallback 規則也改為歸納事件主題，例如：AI/先進封裝、客戶分散、營收財報、法人籌碼、政策風險等。
- 若今日沒有新聞，會明確說沒有抓到今日新事件，不硬寫利多或利空。

## 2. 自選股「當日重點」改成投資組合總覽

- 不是每檔股票各寫一句，而是整理整個自選股組合今天的共同脈絡。
- 固定整理四段：整體表現、主要事件、資金動向、今日關注。
- 會結合自選股漲跌家數、主要拖累/支撐個股、量能放大、新聞事件集中度。
- 若今日新聞不足，改用量價與自選股結構判讀，避免把舊消息包裝成今日事件。

## 3. 「立即更新」按鈕加入轉圈圈

- 按下立即更新後，按鈕會變成「更新中」並顯示 spinner。
- 更新期間按鈕會暫時 disabled，避免重複點擊。
- 立即更新會同步刷新：行情、個股、趨勢圖、新聞與自選股當日重點。

## 4. 保留 V19/V20 原本架構

- 保留 instant / fast search 搜尋邏輯。
- 保留 30 秒背景更新與「更新時不清空畫面」的設定。
- 壓縮檔根資料夾仍是 `taiwan-stock-ai-dashboard-v19`，方便覆蓋原本 GitHub 專案資料夾。
