# V20 更新內容

## 1. 近五天新聞更具體

- 新聞 API 預設帶入 `days=5`。
- Google News RSS 搜尋加入 `when:5d` 條件。
- 新聞項目新增事件分類與 excerpt，例如營收財報、法說展望、訂單客戶、產能投資、產品技術、法人籌碼等。
- 個股 AI 摘要 prompt 改為事件導向，要求先說明近五天實際發生什麼事，不再只套強勢／弱勢模板。

## 2. 自選股當日重點更像事件整理

- `app/api/ai/watchlist/route.ts` 的本地 fallback 改為列出今日實際新聞事件。
- 如果今日沒有新聞，會明確說明沒有抓到當日新聞，再改用當日報價、漲跌幅與成交量整理。
- OpenAI prompt 加強限制：不得捏造標題以外的營收數字、訂單、客戶或公司說法。

## 3. 30 秒背景更新不再把畫面清空

- 台灣加權指數、台指期、行情報價、個股報價、趨勢圖、新聞更新時都保留上一筆資料。
- API 更新失敗時顯示提醒，但不會把畫面切回「載入中」或清空圖表。
- 統計卡、熱門排行、新聞列表與趨勢圖只有第一次沒有資料時才顯示載入中。

## 4. 保留 v19 搜尋架構

- 保留前端本地即時搜尋與後端 fallback 搜尋。
- 不移除原本的 Netlify 設定。
